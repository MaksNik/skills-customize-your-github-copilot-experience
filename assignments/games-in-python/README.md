
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Students will build a classic Hangman game in Python, practicing string manipulation, loops, and conditional logic while handling user input.

## 📝 Tasks

### 🛠️ Game Mechanics

#### Description
Create the core Hangman game so it selects a hidden word, accepts letter guesses, and updates the display as the player progresses.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept letter guesses from the player
- Display the current word progress in `_ _ _` format
- Track and show the number of incorrect guesses remaining

### 🛠️ Win/Lose Flow

#### Description
Implement the game ending conditions and provide clear feedback for success or failure.

#### Requirements
Completed program should:

- End when the word is fully guessed or attempts are exhausted
- Display a win message when the player guesses the word
- Display a lose message when the player runs out of attempts
- Reveal the hidden word after the game ends
