# 📘 Assignment: Introduction to Web Development

## 🎯 Objective

Build a simple interactive web page using HTML, CSS, and JavaScript. This assignment introduces the fundamentals of web development by creating a personal profile page with basic interactivity.

## 📝 Tasks

### 🛠️	Create HTML Structure

#### Description
Create the basic HTML structure for a web page that displays your personal profile information.

#### Requirements
Completed HTML file should:

- Include proper HTML5 document structure with `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags
- Have a `<title>` tag in the head
- Include an `<h1>` heading with your name
- Include at least two paragraphs (`<p>`) with information about yourself
- Include an unordered list (`<ul>`) with at least 3 hobbies or interests
- Include a button element (`<button>`) with text "Click me!"


### 🛠️	Style with CSS

#### Description
Add CSS styling to make your web page visually appealing.

#### Requirements
Completed CSS should:

- Change the background color of the page
- Style the heading with a different font size and color
- Add padding and margins to paragraphs for better spacing
- Style the list items with bullet points and colors
- Style the button with background color, padding, and hover effects


### 🛠️	Add Interactivity with JavaScript

#### Description
Add JavaScript to make the button interactive - when clicked, it should change the text of one of the paragraphs.

#### Requirements
Completed JavaScript should:

- Use `document.querySelector` or `getElementById` to select the button and paragraph elements
- Add an event listener to the button for the 'click' event
- When clicked, change the text content of the paragraph using `textContent` or `innerHTML`
- Include a comment explaining what each part of the code does
